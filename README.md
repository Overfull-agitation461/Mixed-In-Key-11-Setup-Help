Sometimes the installer fails. Use these steps to fix the issue.

### Clear Your Temp Folder
Windows often keeps old files that break new setups. Press the Windows key + R. Type %temp% and hit Enter. Delete all folders and files in this location. Restart your computer and try the setup again.

### Disable Antivirus Software
Some security tools block the installer by mistake. Turn off your real-time protection for ten minutes while you run the installer. Remember to turn the protection back on after the software installs.

### Check For Windows Updates
Outdated Windows files block many modern apps. Open your Settings menu. Select Windows Update and click Check for updates. Install any pending files and restart your machine. This solves most compatibility gaps with audio software.

### User Permissions
Ensure your user profile has full access rights. Log out and sign back in as an administrator. If you share this computer, ask the primary account owner to authorize the install for your profile.
